# 🍟 FAKE Wallet Landing 🪙

¡Bienvenid@ a la landing más fake-tástica del mundo cripto! 🚀  
Este proyecto es una **landing page estática** que simula una plataforma de información y seguimiento de criptomonedas.

⚠️ Importante: No es una wallet real ni maneja datos reales. Es únicamente un proyecto visual / educativo.

---

## 📌 Descripción

FAKE Wallet Landing es una página web desarrollada con **HTML y CSS** que representa el diseño de una plataforma de visualización de mercado cripto.

El objetivo principal del proyecto es:

- Practicar maquetado y diseño web.
- Simular una landing moderna para productos fintech.
- Servir como base para futuros proyectos front-end.
- Usarse como demo visual para portfolio.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Diseño responsive
- Assets gráficos (íconos e imágenes)

No utiliza JavaScript ni backend.
